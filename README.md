# 🚀 Pokedex Vue

This is a simple Pokémon app made with Vue.js and Vuetify.

## 🛠 Tools
- Vue.js
- Vuetify
- Axios
- Vue Router
-  Vuex

## ToDos

## ⚙️ Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Lints and fixes files
```
npm run lint
```
